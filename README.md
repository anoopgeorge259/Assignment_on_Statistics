# 📊 Student Performance Analysis (Statistics Assignment)

This project is part of the ICT Academy of Kerala - Certified Specialist in Data Science and Analytics program.

The analysis is performed using Python in Google Colab on the Students Performance dataset.

---

## 📁 Dataset
The dataset contains student scores in:
- Math
- Reading
- Writing

Along with categorical variables such as:
- Gender
- Lunch type
- Test preparation course
- Parental education level

---

## 🔍 Tasks Performed

### Descriptive Statistics
- Mean, Median, Mode
- Range, Variance, Standard Deviation
- Interquartile Range (IQR)

### Group-wise Analysis
- Scores based on parental education
- Comparison based on test preparation
- Analysis by lunch type

### Correlation & Covariance
- Correlation between:
  - Math & Reading
  - Reading & Writing
- Covariance analysis

### Hypothesis Testing
- **Z-Test**
  - Tested if sample mean math score differs from national average (50)

- **T-Test**
  - Compared reading scores between lunch groups

- **F-Test**
  - Compared variance in math scores based on test preparation

---

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- SciPy
- Google Colab

---

## 📌 Key Insights
- Strong correlation observed between reading and writing scores
- Students with test preparation performed better on average
- Significant difference found in reading scores based on lunch type
- Sample math score mean significantly differs from assumed national average
